# Changelog

All notable changes to CapyBuddy are documented here. This project adheres to
[Semantic Versioning](https://semver.org/).

## [2.0.0] - 2026-06-22

### Changed
- **CapyBuddy is now 100% free and open source — no paywall.** Every feature
  that was previously part of the paid "CapyBuddy Pro" tier is unlocked for
  everyone.
- Merged the separate "CapyBuddy" (App Store) and "CapyBuddy Pro" (website)
  builds into a **single app** with the full feature set, distributed via the
  website with Sparkle auto-updates.

### Removed
- The entire licensing / paywall layer: in-app purchase prompts, the
  Lemon Squeezy license activation, the Keychain-backed entitlement store, and
  the "CapyBuddy Pro" upsell screen.
- The crippled sandboxed App Store target and the runtime `requiresPro`
  feature gating.

### Previously Pro, now free for everyone
- Space Shortcut (hold-Space app launcher chords)
- Screen Recording
- Video Editor
- Snap-to-element screenshots
